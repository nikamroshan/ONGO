# ONGO PWA Icon Generation

Since the automated icon generation is temporarily unavailable, you have two options:

## Option 1: Use Online Icon Generator (Recommended)
1. Visit: https://www.pwabuilder.com/imageGenerator
2. Upload the SVG icon from `public/icons/icon.svg`
3. Download the generated icon pack
4. Extract all icons to `public/icons/` folder

## Option 2: Use Figma/Canva (Manual)
1. Create a 512x512px square design
2. Use green background (#22c55e)
3. Add "ONGO" text in white, bold font
4. Export as PNG in these sizes:
   - 72x72 → icon-72x72.png
   - 96x96 → icon-96x96.png
   - 128x128 → icon-128x128.png
   - 144x144 → icon-144x144.png
   - 152x152 → icon-152x152.png
   - 192x192 → icon-192x192.png
   - 384x384 → icon-384x384.png
   - 512x512 → icon-512x512.png
5. Save all to `public/icons/` folder

## Temporary Solution
For now, I've created an SVG icon that will work as a placeholder. The PWA will still function, but for the best experience, generate proper PNG icons using one of the methods above.
